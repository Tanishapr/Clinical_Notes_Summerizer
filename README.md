# Clinical_Notes_Summerizer
** Overview

This project focuses on the automatic summarization of clinical notes using transformer-based natural language processing models. The system leverages biomedical language representations and sequence-to-sequence models to generate concise and meaningful summaries from unstructured clinical text. The objective is to assist healthcare professionals in quickly understanding patient information and improving decision-making efficiency.

** Features
-> Clinical text summarization using transformer models
-> Integration of Bio_ClinicalBERT for contextual embeddings
-> Use of T5 model for abstractive summarization
-> Evaluation using ROUGE metrics
-> Readability analysis using Flesch Reading Ease score
-> Visualization of embeddings and evaluation metrics
